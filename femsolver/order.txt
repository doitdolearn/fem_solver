1단계: 기반 데이터 구조 (Core Foundation)
수학적 연산이 들어가기 전, 상태를 저장하고 공간을 정의하는 가장 단순한 클래스들부터 작성합니다.

femsolver/core/mesh.py: 물리적 계산이 없는 기하학적 뼈대인 Node, Element, Mesh 클래스를 구현합니다.

femsolver/materials/: 물성치 컨테이너인 LinearElasticMaterial과 ThermalIsotropicMaterial을 작성합니다.

femsolver/utils/quadrature.py: 추후 2D 요소 적분에 필수적인 가우스 구적법(Gauss-Legendre) 테이블과 함수를 미리 구현해 둡니다.

2단계: 어셈블리 엔진 (Assembly & DOFs)
단일 요소를 전체 시스템의 행렬로 조립하기 위한 솔버의 '심장'을 만듭니다.

femsolver/core/dof_manager.py: (노드 번호, 자유도 이름)을 1차원 글로벌 인덱스로 변환하는 핵심 매핑 로직을 작성합니다.

femsolver/core/boundary_conditions.py: DirichletBC와 NodalLoad 등 경계조건을 담을 데이터 클래스를 정의합니다.

femsolver/core/assembler.py: 요소에서 계산된 국소 행렬(k_e)을 받아 scipy.sparse.coo_matrix를 활용해 글로벌 강성 행렬(K)로 조립하는 로직을 구현합니다.

femsolver/utils/linalg_utils.py: 페널티 메서드 대신 정확도가 높은 파티션 메서드(partition_system)를 이용한 경계조건 부여 로직을 작성합니다.

3단계: 기본 요소 구현 (MVP Elements)
전체 시스템을 테스트하기 위해 가장 단순하고 직관적인 1D 요소를 먼저 완성합니다.

femsolver/elements/base.py & registry.py: 모든 요소가 지켜야 할 추상 기본 클래스(ABC)를 정의하고 레지스트리 패턴을 설정합니다.

femsolver/elements/bar1d.py: 2노드 1D 축 요소의 강성 행렬(EA/L)과 하중 벡터를 하드코딩 수준으로 간단하게 구현합니다.

4단계: 첫 물리 문제와 솔버 (End-to-End Test)
파일 입출력 없이 코드상에서 하드코딩된 객체들만으로 K·u = F를 푸는 최소 기능 제품(MVP)을 완성합니다.

femsolver/solvers/direct.py: scipy.sparse.linalg.spsolve를 감싸는 간단한 래퍼를 만듭니다.

femsolver/physics/base_problem.py & structural_static.py: 위에서 만든 Assembler와 DOFManager를 호출해 행렬을 조립하고, 파티션 메서드로 경계조건을 적용한 뒤 해를 구하는 파이프라인을 작성합니다.

femsolver/postprocessing/reaction_forces.py: 고정점에서의 반력을 계산하는 로직을 추가합니다.

5단계: 입출력 파이프라인 (I/O Pipeline)
솔버가 실제 소프트웨어처럼 동작하도록 YAML 파싱과 CLI 엔트리 포인트를 붙입니다.

femsolver/io/yaml_reader.py & input_validator.py: 외부 라이브러리인 pyyaml을 이용해 데이터를 읽고 스키마를 검증합니다.

femsolver/io/model_builder.py: 검증된 딕셔너리 데이터를 1~2단계에서 만든 파이썬 객체(Mesh, DOFManager 등)로 변환하는 팩토리 역할을 구현합니다.

femsolver/io/yaml_writer.py: 계산된 SolveResult를 다시 YAML 형태로 출력합니다.

solve.py: 터미널에서 python solve.py -i input.yaml -o output.yaml 명령어가 동작하도록 argparse를 연결합니다.

6단계: 2D 확장 (2D Elements)
차원을 넓히고 내부 응력 계산 기능을 추가합니다.

femsolver/elements/bar2d.py: 좌표 변환 행렬(Coordinate Transform)을 적용해 2D 트러스 요소를 만듭니다.

femsolver/elements/tri3.py & quad4.py: B-matrix와 1단계의 구적법(quadrature)을 활용해 2D 평면 요소들을 구현합니다.

femsolver/postprocessing/stress_recovery.py: 각 요소의 변위를 이용해 응력(von Mises 등)을 복원하는 로직을 붙입니다.

7단계: 고급 물리 모델 확장 (Advanced Physics)
정적 해석 구조가 탄탄해졌으므로 다른 지배방정식을 가진 해석들을 하나씩 병렬로 추가합니다.

열 해석: thermal_steady.py 및 관련 열전달 요소(rod1d_thermal, tri3_thermal).

동 해석: structural_dynamic.py와 질량 행렬 조립, 그리고 newmark.py 시간 적분기.

모달 해석: 고유값 문제를 위한 modal.py와 eigenvalue.py.

비선형 해석: 하중 증분과 newton_raphson.py, 그리고 비선형 요소(bar1d_nl.py).

8단계: 시각화 및 마무리 (Visualization)
최종적으로 사용자 경험을 완성하는 후처리 모듈을 작성합니다.

femsolver/postprocessing/vtk_writer.py: ParaView 등에서 볼 수 있도록 순수 XML 형태의 .vtu 출력 로직을 만듭니다.

femsolver/postprocessing/plot_results.py: Matplotlib을 활용해 변형 형상이나 응력 분포를 그려주는 함수를 작성합니다.

visualize.py: 터미널에서 플롯을 띄우거나 이미지를 저장하는 CLI 스크립트를 완성합니다.

본격적인 코딩에 들어가기 전, 특정 부분의 구현 방식을 짚고 넘어갈 수 있습니다.
